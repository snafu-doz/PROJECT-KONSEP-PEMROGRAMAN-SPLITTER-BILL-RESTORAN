Penjelasan Project : 
Splitter Bill Restoran adalah program berbasis bahasa C yang dirancang untuk mempermudah pembagian tagihan makan di restoran. Program ini menerima input menu dan harga, menghitung pajak 10% dan service charge 5%, lalu menawarkan dua metode pembagian: sama rata dengan pembulatan ke kelipatan 1000, atau custom sesuai keinginan pengguna. Dengan antarmuka teks yang sederhana, program ini membantu menghilangkan kesulitan perhitungan manual dan mengurangi potensi kesalahan dalam membagi tagihan.
